# Tier 01 — Basics

**Status: implemented.** These programs parse against the current grammar.

## Covers
- `let name = value` bindings
- Integer literals
- Arithmetic `+` and `*`, with `*` binding tighter than `+`
- Parenthesised grouping
- `//` line comments
- Newline-terminated statements (no semicolons)

## Prerequisites
None — this is the floor of the language.
