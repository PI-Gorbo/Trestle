# Tier 04 — Generics

Abstraction over types. Builds on functions (tier 00) and records/ADTs (tier 03).

- `generic-functions` — functions abstracted over a type parameter, e.g.
  `<T>(x: T) => x`.
- `higher-order-data-types` — a data type parameterised by another type, e.g.
  `Option<T>`.

All ignored *(proposed syntax)*.
